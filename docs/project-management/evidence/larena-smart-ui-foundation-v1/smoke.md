# Smoke

Package smoke on PHP 8.4 confirms that the default registry produces ten
reference-ready entries in English and Russian in stable order, and that two AI
JSON serializations are byte-identical. Every entry renders its real pinned
custom element through SmartManager with an exact manifest asset contract.
`AdminDataviewRenderer` produces a real `<sf-table>` artifact through
`ui.dataview` and preserves escaped hydration data.

All 48 controls were changed one at a time through the submitted reference
contract and produced different resolved props and host HTML. Dropdown options
render as bounded `<sf-list-item>` children; dataview data renders through the
escaped hydration payload. Neither structured value is flattened into a false
frontend attribute example.

Protected Admin route and browser acceptance remain in the integration batch;
this package smoke does not substitute for them.
