# Larena UI

Developer vertical slice: Larena-native Button, Badge, Toolbar, EmptyState, Pagination manifests and the backend-first `admin.dataview` Smart Component are used by real admin screens with versioned assets activated through `core.assets`. This is not a full SF5 runtime or production-readiness claim.

Universal UI/frontend package for Larena smart components, backend-first rendering, hydration, design packs, UI asset graph, component registry, playground and frontend behavior contracts.

This repository is a clean Larena pre-codegen baseline. Package implementation code has not started.

Canonical specifications are in `simai/larena-specs`.
